# WAYBACK MACHINE

# Introduction

아카이빙(archiving)하고자 하는 웹 사이트(url)를 지정하여 매주 월요일 10시에 스크래이핑하는 기능을 구현해 보았습니다. /
검색한 웹 사이트의 아카이빙 기록이 있으면 달력에 표시가 되도록 하였습니다.
